    Shopping List!

    Please note any changes can be altered by other users

    How to setup as an app

    iOS:
    Share > Add to Home Screen > Add

    Android:
    Menu Icon (top right-hand corner) > Add to Home Screen > Add

    How to use

    To add an item, type it in and either press 'enter' on the keyboard or 'add to cart' button to add to the list

    To delete an item, double tap it

    After deleting an item, it will be copied on your device, so if you want to recover or edit what's written, you can paste and re-enter it

    Please note that this app does require an internet connection to sync and update any changes

    About

    This is a web-app; it is not the same as a native app like WhatsApp or Instagram, where you download software on your device

    It is a responsive website that behaves as an app on your device when you add it to the home screen

    This means that it will act as a regular app that requires an internet connection, but without taking up space on your device's storage

    It can be used by an infinite amount of people simultaneously and they will be making changes to the same shopping list as you, which is perfect when using within households

    However, be careful as this means that everyone using the same URL of this app will be modifying the same list

    Sharing your URL

    When you share your URL with someone, they will have the same editing access as you

    If the URL is shared among too many people, and they are making unwanted changes, please contact me and I can change it for you at any time

    This will stop everyone currently using this app and you will be able to re-add the new URL to your home screen

    Please note, changing the URL will not affect the current contents of the list - they will stay as they are, unless someone with access has deleted them

    No internet?

    If you are going to be in a location without an internet connection, open the app in an area with connection beforehand, and leave it running in the background

    You will be able to view your shopping list and delete items, so you can tick off items as you shop, however when the app is reloaded, the items will still be there as the app cannot sync while the device is offline

    You can screenshot the list before reloading the app, or use it as a checklist after your shop to make sure you got everything you need
