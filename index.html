    How to Setup as an App

    iOS:
    Open in Safari > Share > Add to Home Screen > Add

    Android:
    Menu Icon (top right-hand corner) > Add to Home Screen > Add

    How to Use

    To add an item, type it and press 'Enter' or click 'Add to Cart'

    Tap an item to adjust quantity, view note (if applicable), add to the alternate list, or edit it.

    To delete an item, first edit it and then click the trash icon to remove it permanently.

    Items can be organized into sections, such as by shop or food type, allowing you to easily categorize and view them based on your preferences.

    This app requires an internet connection to sync changes.

    Key Features

    Store regular items in a separate list for easy addition later.

    Organize items into custom categories (e.g. by shop or type of food).

    Add notes to items for reminders.

    Edit or permanently delete items using the new menu interface.

    About

    This is a web-based app that functions like a native app when added to your home screen. It does not take up space on your device and requires an internet connection to sync across devices.

    Everyone with the same URL can edit the list. Be cautious when sharing the link to avoid unwanted changes.

    Offline Use

    Open the app in an area with internet to load your list, and you can use it offline for viewing and deleting and items. However, the app cannot sync changes while offline.

    You can screenshot your list to use it as a checklist.
